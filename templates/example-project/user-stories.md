# 用户故事

## US-001 用户注册
**作为** 一个新用户，
**我希望** 通过邮箱注册账号，
**以便** 我可以使用系统的功能。

**验收标准：**
- 输入有效邮箱和密码后创建账号
- 密码不满足强度要求时提示具体规则
- 注册成功后发送验证邮件

## US-002 用户登录
**作为** 已注册用户，
**我希望** 通过邮箱和密码登录，
**以便** 访问我的个人数据和功能。

**验收标准：**
- 输入正确凭据后跳转到主页
- 错误凭据显示通用错误信息
- 连续5次失败后账户锁定15分钟

## US-003 权限分配
**作为** 管理员，
**我希望** 为用户分配角色，
**以便** 控制不同用户的操作权限。

**验收标准：**
- 管理员可为用户分配 admin/editor/viewer 角色
- 角色变更立即生效
- 操作记录写入审计日志
