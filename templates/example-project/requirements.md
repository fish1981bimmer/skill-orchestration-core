# 需求分析

## Overview

本文档描述示例项目的功能需求，用于指导后续开发。

## 功能需求

### FR-001 用户注册
- 支持邮箱和手机号注册
- 密码强度校验（至少8位，含大小写和数字）
- 注册成功发送验证邮件

### FR-002 用户登录
- 支持邮箱/手机号 + 密码登录
- 支持 OAuth2 第三方登录（GitHub/Google）
- 登录失败5次后锁定15分钟

### FR-003 权限管理
- RBAC 角色模型：admin / editor / viewer
- 资源级别权限控制
- 操作审计日志

## 非功能需求

- 响应时间 < 200ms (P95)
- 支持 1000 并发用户
- 数据加密存储
